## 📱 使用流程

### 添加授权

1. 进入"个人中心" → "设置"
2. 找到"授权管理"模块
3. 点击"+ 添加授权"按钮
4. 搜索并选择医生
5. 设置授权范围和数据范围
6. 选择授权有效期
7. 勾选"同意《数据授权协议》"
8. 确认授权

### 管理授权

**查看详情**
- 点击授权卡片上的"查看详情"
- 查看完整授权信息和访问统计

**修改权限**
- 点击"修改权限"按钮
- 调整授权范围或有效期（待实现）

**撤销授权**
- 点击"撤销授权"按钮
- 确认后立即生效
- 医生无法再访问患者数据

## 🔒 安全特性

### 1. 法律合规

✅ **明确告知**：授权前清晰展示授权内容和范围
✅ **自愿同意**：患者主动选择授权
✅ **可撤销**：患者可随时撤销授权
✅ **目的限制**：明确说明仅用于随诊健康管理
✅ **数据最小化**：只授权必要的数据类型和时间范围

### 2. 权限控制

✅ **授权立即生效**：无需审批流程
✅ **医生收到通知**：授权后立即发送通知
✅ **访问审计**：记录每次访问（待实现）
✅ **数据脱敏**：返回给医生的数据需脱敏处理（待实现）

### 3. 用户体验

✅ **简洁流程**：3步完成授权
✅ **清晰提示**：每步都有明确提示
✅ **确认机制**：撤销授权需二次确认
✅ **历史记录**：可查看授权历史

## 📝 使用示例

### 示例 1：添加新授权

```typescript
// 在 AuthorizationManagement 组件中
const handleAdd = (data: any) => {
  const newAuth: DoctorAuthorization = {
    id: `auth${Date.now()}`,
    doctorId: data.doctorId,
    doctorName: data.doctorName,
    hospital: data.hospital,
    department: data.department,
    licenseNumber: data.licenseNumber,
    patientId: 'patient1',
    authorizationType: data.authorizationType,
    authorizedAt: new Date().toISOString(),
    expiresAt: data.expiryDays === 99999
      ? undefined
      : new Date(Date.now() + data.expiryDays * 24 * 60 * 60 * 1000).toISOString(),
    status: 'active',
    scope: {
      startDate: new Date().toISOString().split('T')[0],
      dataRange: data.dataRange,
    },
    accessCount: 0,
  };
  setAuthorizations((prev) => [...prev, newAuth]);
};
```

### 示例 2：撤销授权

```typescript
const handleRevoke = (id: string) => {
  if (confirm('确定要撤销此授权吗？')) {
    setAuthorizations((prev) =>
      prev.map((a) =>
        a.id === id ? { ...a, status: 'revoked' as const } : a
      )
    );
  }
};
```

## 🎯 核心设计原则

1. **用户自主控制**：患者完全掌控授权
2. **立即生效**：无需审批，符合用户预期
3. **简单易用**：3步完成授权
4. **安全合规**：符合《个人信息保护法》要求
5. **透明可溯**：完整的授权和访问记录

## 🆘 常见问题

**Q: 授权后医生能立即查看数据吗？**
A: 是的，授权立即生效，医生收到通知后可直接访问。

**Q: 患者可以撤销授权吗？**
A: 可以，患者可随时在"授权管理"中撤销授权，撤销后医生无法继续访问。

**Q: 授权数据会包含哪些内容？**
A: 只包含患者明确授权的数据类型（餐食记录、健康报告、对话日志），不包含敏感个人信息（住址、身份证号等）。

**Q: 医生可以分享授权数据吗？**
A: 不可以，数据仅用于随诊健康管理，不得用于其他目的。

**Q: 授权到期后医生还能查看数据吗？**
A: 不能，授权到期后医生无法继续访问患者数据。
